# GIT & GITHUB

bu projede git & github temel olarak ogrendiklerimi yukledigim repodur.

## Git & Github Farklılıkları

### Git

- Version Kontrol Sistemidir
- Önceki versionlara dönebilir
- Projenin gelişimini takip eder
- Proje yedeklerinin alınması
- Ekip çalışması uyumluluğu
- Dosyaların neden ve nerede değiştiğini anlama ve takip edebilme

### Github

- En bilinen proje saklama platformudur
- Git aracılığıyla projeyi depolandığı yerdir
- CV değeri görür
- Yazılımcıların ilham alabileceği ve destek görebileceği bir yerdir
- Topluluk sayesinde sorunlarının çözüme kavuşabildiği platformudur

## Git ve Github Kimin İçin

- Platformdan bağımsızdır, programcı ve geliştiriciler içindir
