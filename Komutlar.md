# KOMUTLAR

PWS: powershell komut sistemi
CMD: windows komut sistemi

- pwd: dizinin yolunu gösterir
- ls: dizinin içindekilerini gösterir
- ls -a: dizinin gizli içerikleri dahil içindekilerini gösterir
- clear: terminali temizler (PWS)
- cls: terminali temizler (CMD)
- cd: klasörlere girmeyi sağlar
- mkdir: klasör oluşturur
